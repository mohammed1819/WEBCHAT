# PulseBridge

PulseBridge is a real-time messaging dashboard built with React, Node.js, Express, MongoDB, and Socket.IO. It includes user authentication, profile status updates, image attachments, and live contact presence.

## Features

- Real-time one-to-one chat with Socket.IO
- User authentication with JWT cookies
- Profile picture upload via Cloudinary
- Contact search and presence status
- Editable profile status message
